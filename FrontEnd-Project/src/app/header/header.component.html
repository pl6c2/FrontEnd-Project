As mentioned above, ‘online host, visualization and deployment’ gives the user a platform where user can upload or download the models or pretty much anything. User can view any model as a separate module and can comment and rate the model.

We are following the MEAN stack approach using Angular as front-end application framework, to create the views for user interaction, Node js and Express to handle the business logic and establish the connectivity to database, for user validations and retrieving the details regarding the models. The Discussion form for each issue is going to have a thread of comments and other features which are planning to be implemented.
